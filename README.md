## 📌 Overview
This repository contains the complete collection of 5 Enterprise-Grade Artificial Intelligence and Machine Learning projects developed during my 2-week intensive internship at **Elevate Labs**. 

Each project focuses on solving real-world industry problems using advanced concepts in Deep Learning, Natural Language Processing (NLP), and Time-Series Forecasting, fully deployed as interactive web applications.

---

## 📂 Project Directory (As per Internship Guidelines)

### 1. Stock Price Trend Prediction with LSTM (Project #3)
* **Description:** A Time-Series forecasting dashboard that fetches live market data using the `yfinance` API and utilizes a Deep Learning Long Short-Term Memory (LSTM) Neural Network to predict future stock price movements.
* **Repository:** [MarketVision-Stock-Predictor](https://github.com/mdsalmanfarsi692004-svg/MarketVision-Stock-Predictor)

### 2. AI-Powered Resume Ranker (Project #4)
* **Description:** An Enterprise-Grade NLP application engineered to automate the recruiter screening process. It ingests raw resumes and compares them against target Job Descriptions using TF-IDF and Cosine Similarity.
* **Repository:** [AI-Resume-Ranker](https://github.com/mdsalmanfarsi692004-svg/AI-Resume-Ranker)

### 3. Movie Recommendation System (Project #8)
* **Description:** A sophisticated content-based recommendation engine modeled after premium OTT platforms. It analyzes movie genres and plot descriptions using NLP to recommend mathematically relevant movies.
* **Repository:** [CineMatch-Movie-Recommender](https://github.com/mdsalmanfarsi692004-svg/CineMatch-Movie-Recommender)

### 4. Fraud Detection in Credit Card Transactions (Project #10)
* **Description:** A high-performance predictive cybersecurity model built to detect fraudulent credit card transactions in real-time, utilizing the powerful XGBoost algorithm to handle highly imbalanced financial datasets.
* **Repository:** [FraudGuard-AI-Anomaly-Detection](https://github.com/mdsalmanfarsi692004-svg/FraudGuard-AI-Anomaly-Detection)

### 5. AI Dungeon Story Generator (Project #15)
* **Description:** An advanced Natural Language Generation (NLG) platform leveraging cutting-edge Transformer models from Hugging Face to generate coherent, abstractive, and interactive fantasy stories.
* **Repository:** [AI-Dungeon-Pro](https://github.com/mdsalmanfarsi692004-svg/AI-Dungeon-Pro)
